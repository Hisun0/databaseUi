# DatabaseUI

Данный проект позволяет манипулировать базой данных через простой интерфейс. В проекте используются основные CRUD операции (get, post, patch, delete)

## Стек технологий

### Фронтенд:

1. React, React Router, Redux Toolkit
2. Typescript
3. SCSS (flex, grid)

### Бекенд:

1. Express.js
2. Typescript
3. MongoDB
